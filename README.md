This page is an interactive quiz for kids to practice English.

First, the user needs to press the button.

<img src="https://cdn.discordapp.com/attachments/1016745958727491615/1035568232301609122/1.png" />

Then, a random word will appear below the machine. Clicking on the button plays the sound of the word.

<img src="https://cdn.discordapp.com/attachments/1016745958727491615/1035568232708444180/2.png" />

The user will choose the image. If the answer is incorrect, the phrase "Try again!" will be displayed.

<img src="https://cdn.discordapp.com/attachments/1016745958727491615/1035568233123696640/3.png" />

If the answer is correct, the text will change to "Well done!". A fun fact about the word will also be displayed.

<img src="https://cdn.discordapp.com/attachments/1016745958727491615/1035568230762303498/4.png" />

The user can click the "restart" button to refresh the page and restart the game, which is the end of the flow.

All the elements are playful, interactive and easy to use, so that kids can learn and have fun.

PS: The page contains animated elements that blink.
